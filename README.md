# Zabbup - Configuration backup utility for Zabbix entities

* Features
  * Backups via Zabbix API configuration.export of
    * Hosts
    * Host groups
    * Templates
    * Template groups
    * Maps
    * Mediatypes
    * Images
  * Export to
    * Remote git repository
  * Encryption

* Roadmap
  * CLI arguments for
    * zabbup-backup
    * zabbup-decrypt
  * Documentation :D
  * Additional metadata for each export file to check if encrypted data changed
  * Export to S3
